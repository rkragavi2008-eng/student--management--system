# Student Management System

A simple C++ project that manages student records using file handling.

##  Features
- Add Student Details
- Display All Students
- Search Student
- Update Student Information
- Delete Student Record

##  Technology Used
- C++

## How to Run

1. Compile the program:
   g++ main.cpp -o student

2. Run the program:
   ./student

##  Project Description
This project is a basic console-based application that helps in managing student data efficiently. It demonstrates the use of file handling in C++.
